## ğŸ¯ Key Features

### 1. Hero Section

- **Gradient Background**: Blue gradient for visual appeal
- **Bilingual**: English + Thai church name
- **CTAs**: Two prominent action buttons

### 2. Feature Cards (4-Grid)

- **Icons**: Visual indicators for each section
- **Color-Coded**: Blue, Green, Purple, Orange
- **Responsive**: Stacks on mobile, 2-col tablet, 4-col desktop

### 3. Events Section

- **Cards**: Event details with badges
- **Icons**: Calendar and Clock for context
- **CTA**: Links to full calendar

### 4. Sermons Carousel

- **Interactive**: Swipeable/clickable navigation
- **Cards**: Speaker, date, scripture info
- **Badges**: Duration indicators

### 5. Welcome Section

- **Two-Column**: Text + Image layout
- **Personal**: Pastor introduction
- **Inviting**: Warm welcome message

### 6. Newsletter

- **Email Capture**: Simple subscription form
- **Branded**: Matches hero color scheme

### 7. Footer

- **Four Columns**: Organized information
- **Complete Info**: Times, links, contact
- **Professional**: Dark theme with good contrast

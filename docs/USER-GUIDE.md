## Getting Started

### Logging In

1. Visit the application at your church's URL
2. Click "Sign In" on the landing page
3. Enter your email and password
4. If MFA is enabled, enter your verification code

### First-Time Setup

After your account is created by an administrator:

1. Check your email for login credentials
2. Log in with the provided password
3. Change your password immediately
4. (Optional) Enable Two-Factor Authentication for extra security

## Events

### Viewing Events

1. Click "Events" in the navigation menu
2. Browse upcoming events
3. Click on any event for full details

### RSVPing to Events

1. Open an event
2. Click "RSVP" to confirm attendance
3. Select your response (Yes/No/Maybe)
4. Optionally add the number of guests

### Your RSVPs

View all your RSVPs:

1. Go to Dashboard
2. Click "My RSVPs" section
3. See all events you've responded to

## Troubleshooting

### Can't Log In?

1. Verify your email is correct
2. Click "Forgot Password" to reset
3. Contact administrator if issues persist

### Not Receiving Notifications?

1. Check notification settings
2. Ensure browser permissions are granted
3. Try disabling and re-enabling notifications

### Page Not Loading?

1. Refresh the page
2. Clear browser cache
3. Try a different browser
4. Contact support if issue persists

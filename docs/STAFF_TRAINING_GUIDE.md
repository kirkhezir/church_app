## Getting Started

### First Login

1. Open your web browser and go to **[your-church-app-url]**
2. Enter your email address and password
3. Click **Login**
4. If prompted, set up Two-Factor Authentication (2FA) for added security

### Setting Up Your Profile

1. Click your name in the top right corner
2. Select **Profile**
3. Update your information:
   - Phone number
   - Address
   - Privacy settings (who can see your info)
4. Click **Save Changes**

### Enabling Security (MFA)

1. Go to **Settings > Security**
2. Click **Enable Two-Factor Authentication**
3. Scan the QR code with your authenticator app (Google Authenticator, Authy, etc.)
4. Enter the 6-digit code to confirm
5. **Important**: Save your backup codes in a safe place!

## Managing Members

### Viewing Members

1. Click **Members** in the sidebar
2. Browse the member directory
3. Use the search bar to find specific members
4. Filter by role (Admin, Staff, Member)

### Adding a New Member

1. Click **Add Member** button
2. Fill in required information:
   - Email address
   - First and last name
   - Phone number (optional)
   - Membership date
3. Select their role
4. Click **Create Member**
5. The system will send them a welcome email with login instructions

### Editing Member Information

1. Click on a member's name
2. Click **Edit** button
3. Update the necessary fields
4. Click **Save Changes**

### Managing Roles

| Role       | Can Do                                          |
| ---------- | ----------------------------------------------- |
| **Admin**  | Everything - full system access                 |
| **Staff**  | Manage events, announcements, view analytics    |
| **Member** | View events, RSVP, read announcements, messages |

## Managing Events

### Creating an Event

1. Click **Events** in the sidebar
2. Click **Create Event**
3. Fill in the event details:
   - **Title**: Clear, descriptive name
   - **Description**: What the event is about
   - **Date & Time**: When it starts and ends
   - **Location**: Where it will be held
   - **Category**: Worship, Bible Study, Community, Fellowship
   - **Capacity**: Maximum attendees (optional)
   - **Image**: Upload an event image (optional)
4. Click **Create Event**

### Managing RSVPs

1. Open an event
2. View the **RSVP** tab
3. See who's attending:
   - ✅ Confirmed
   - ⏳ Waitlisted (if over capacity)
   - ❌ Cancelled
4. Export attendee list if needed

### Editing Events

1. Open the event
2. Click **Edit**
3. Make changes
4. Click **Save**
5. Attendees will be notified of changes

### Cancelling Events

1. Open the event
2. Click **Cancel Event**
3. Provide a reason (optional)
4. Confirm cancellation
5. All attendees will be notified automatically

## Announcements

### Creating an Announcement

1. Click **Announcements** in the sidebar
2. Click **Create Announcement**
3. Enter:
   - **Title**: Brief headline
   - **Content**: Full announcement text
   - **Priority**: Normal or Urgent
   - **Expiry Date**: When to stop showing (optional)
4. Click **Publish**

### Announcement Priority

- **Normal**: Regular announcements, shown in standard list
- **Urgent**: Highlighted prominently, members receive push notifications

### Viewing Announcement Stats

1. Open an announcement
2. View the **Stats** tab
3. See:
   - How many members viewed it
   - When they viewed it
   - Read rate percentage

## Messages

### Sending a Message

1. Click **Messages** in the sidebar
2. Click **New Message**
3. Select recipient(s)
4. Type your message
5. Click **Send**

### Reading Messages

1. Click on a conversation
2. View the message thread
3. Type a reply at the bottom
4. Press Enter or click Send

### Message Notifications

- You'll receive push notifications for new messages (if enabled)
- Email notifications sent for unread messages

## Mobile App Usage

### Installing the App

**On iPhone (Safari):**

1. Open the app in Safari
2. Tap the Share button (box with arrow)
3. Tap "Add to Home Screen"
4. Tap "Add"

**On Android (Chrome):**

1. Open the app in Chrome
2. Tap the menu (three dots)
3. Tap "Add to Home Screen"
4. Tap "Add"

### Offline Usage

- The app works offline with limited features
- View cached events and announcements
- Actions sync when you're back online

### Push Notifications

1. Allow notifications when prompted
2. Or enable in Settings > Notifications
3. Types of notifications:
   - Event reminders (24h and 1h before)
   - New announcements
   - New messages

## Troubleshooting

### Can't Log In

1. Check your email address is correct
2. Use "Forgot Password" to reset
3. Clear browser cache and try again
4. Contact an admin if still having issues

### Not Receiving Notifications

1. Check device notification settings
2. Enable notifications in app settings
3. Make sure you're not in Do Not Disturb mode

### Page Won't Load

1. Check your internet connection
2. Try refreshing the page
3. Clear browser cache
4. Try a different browser

### Event Not Showing

1. Check the event date hasn't passed
2. Ensure you have permission to view it
3. Try refreshing the page
